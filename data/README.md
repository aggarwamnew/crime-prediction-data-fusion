# Data Acquisition Guide

Raw data is not included in this repository due to size (approximately 5 GB) and licensing considerations. All sources are published under the Open Government Licence or equivalent open licences.

This guide provides step-by-step instructions for downloading each dataset.

## Directory Structure

After downloading, your `data/` directory should look like:

```
data/
├── raw/
│   └── london/
│       ├── crime/              # Monthly CSV files from data.police.uk
│       ├── boundaries/         # LSOA 2021 boundary shapefiles
│       ├── imd/                # IMD 2019 and 2025 Excel/CSV files
│       ├── census/
│       │   ├── ts003/          # Household composition
│       │   ├── ts006/          # Population density
│       │   ├── ts007a/         # Age by single year
│       │   └── ts067/          # Highest qualification
│       ├── weather/            # Met Office monthly observations
│       ├── housing/            # ONS HPSSA median prices
│       ├── pois/               # OpenStreetMap POI extracts
│       ├── mental_health/      # SAMHI index CSV
│       ├── school_holidays/    # Term date schedules
│       └── transport/
│           ├── ptal/           # Public Transport Accessibility Level
│           ├── station_taps/   # TfL daily entry/exit counts
│           └── santander_cycles/  # Bike hire trip records
└── processed/
    └── london/                 # Generated by scripts 01-02
```

## Step-by-Step Downloads

### 1. Crime Data (Tier 1: Baseline)

**Source:** data.police.uk (Home Office)

1. Go to https://data.police.uk/data/
2. Select forces: **Metropolitan Police Service** and **City of London Police**
3. Select date range: **February 2023 to January 2026** (36 months)
4. Download and extract CSVs to `data/raw/london/crime/`
5. Expected: approximately 36 monthly CSV files, totalling around 500 MB

### 2. LSOA Boundaries

**Source:** ONS Open Geography Portal

1. Download LSOA 2021 boundaries (BGC = clipped to coastline):
   https://geoportal.statistics.gov.uk/datasets/ons::lsoa-dec-2021-boundaries-generalised-clipped-ew-bgc/
2. Download as GeoJSON or Shapefile
3. Save to `data/raw/london/boundaries/`

### 3. Index of Multiple Deprivation (Tier 2: Contextual)

**Source:** Ministry of Housing, Communities and Local Government

- **IMD 2019:** https://www.gov.uk/government/statistics/english-indices-of-deprivation-2019
  - Download "File 7: All ranks, deciles and scores for all indices for all LSOAs"
- **IMD 2025:** https://www.gov.uk/government/statistics/english-indices-of-deprivation-2025
  - Download the equivalent file

Save to `data/raw/london/imd/`

### 4. Weather (Tier 2: Contextual)

**Source:** Met Office MIDAS Open via CEDA

1. Register at https://catalogue.ceda.ac.uk/ (free academic account)
2. Search for London Heathrow station observations
3. Download monthly summary data for the study period
4. Save to `data/raw/london/weather/`

Key variables: mean daily max/min temperature, mean temperature, total rainfall, total sunshine hours.

### 5. Demographics (Tier 2: Contextual)

**Source:** ONS Census 2021 via Nomis

- **Population density (TS006):** https://www.nomisweb.co.uk/output/census/2021/census2021-ts006.zip
- **Age by single year (TS007a):** https://www.nomisweb.co.uk/output/census/2021/census2021-ts007a.zip

Download and extract to `data/raw/london/census/ts006/` and `data/raw/london/census/ts007a/`.

### 6. Points of Interest (Tier 2: Contextual)

**Source:** OpenStreetMap via Overpass API

Script `12_poi_extraction.py` handles this automatically. It queries the Overpass API for 10 amenity categories within Greater London:

- Pubs, bars, ATMs, fast food, nightclubs, restaurants, schools, bus stops, parks, shops

Output is saved to `data/raw/london/pois/`. Alternatively, use the Overpass Turbo web interface: https://overpass-turbo.eu/

### 7. Housing Prices (Tier 2: Contextual)

**Source:** ONS House Price Statistics for Small Areas (HPSSA)

1. Download Dataset 9 (median price paid by LSOA):
   https://www.ons.gov.uk/peoplepopulationandcommunity/housing/datasets/medianhousepricefornationalandsubnationalgeographiesquarterlyrollingyearhpssadataset09
2. Save to `data/raw/london/housing/`

### 8. School Holidays and Bank Holidays (Tier 3: Temporal)

**Source:** gov.uk

- School term dates: https://www.gov.uk/school-term-and-holiday-dates
- Bank holidays: https://www.gov.uk/bank-holidays

Compile London state school holiday periods and bank holiday dates for Feb 2023 to Jan 2026 into CSV format. Save to `data/raw/london/school_holidays/`.

### 9. Mental Health Index (Tier 4: Socio-Structural)

**Source:** Place-based Longitudinal Data Resource (PLDR)

1. Download SAMHI v5.00 (2011-2022, LSOA level):
   https://pldr.org/dataset/small-area-mental-health-index-samhi-2noyv
2. Direct CSV download:
   https://pldr.org/download/2noyv/q3n/samhi_21_01_v5.00_2011_2022_LSOA.csv
3. Save to `data/raw/london/mental_health/`

### 10. Education Attainment (Tier 4: Socio-Structural)

**Source:** ONS Census 2021

- **TS067 (Highest qualification):** https://www.nomisweb.co.uk/output/census/2021/census2021-ts067.zip

Extract to `data/raw/london/census/ts067/`.

### 11. Household Composition (Tier 4: Socio-Structural)

**Source:** ONS Census 2021

- **TS003 (Household composition):** https://www.nomisweb.co.uk/output/census/2021/census2021-ts003.zip

Extract to `data/raw/london/census/ts003/`.

### 12. PTAL (Transport)

**Source:** GLA Datastore

1. Download Public Transport Accessibility Level data:
   https://data.london.gov.uk/dataset/public-transport-accessibility-levels
2. Save to `data/raw/london/transport/ptal/`

### 13. TfL Station Footfall (Transport)

**Source:** TfL Open Data

1. Register at https://api-portal.tfl.gov.uk/
2. Download daily entry/exit counts for London Underground, Overground, and Elizabeth Line stations
3. Save to `data/raw/london/transport/station_taps/`

### 14. Santander Cycles (Transport)

**Source:** TfL Cycling Data

Use the provided download script:

```bash
python scripts/download_santander.py
# or
bash scripts/download_santander_trips.sh
```

This downloads approximately 40 million trip records from 799 docking stations.
Files are saved to `data/raw/london/transport/santander_cycles/`.

## Cross-City Data (Chicago and Vancouver)

The Chicago and Vancouver replications automate their own downloads: each `01_download_crime.py` and `02_download_boundaries.py` (plus the per-layer scripts) pull directly from the public APIs into `data/raw/chicago/` and `data/raw/vancouver/`. No manual download is required. Full source URLs, dataset IDs, and the open-versus-gated analysis are in [`cross_city_feasibility.md`](cross_city_feasibility.md).

**Chicago** (`notebooks/chicago/`):

| Layer | Source | Access |
|-------|--------|--------|
| Crime | Chicago Data Portal `ijzp-q8t2` (Socrata API) | scripted |
| Boundaries | US Census TIGER/Line 2020 tracts (Cook County) | scripted |
| Deprivation | CDC/ATSDR Social Vulnerability Index 2022 | scripted |
| Weather | NOAA GHCN-Daily (O'Hare `USW00094846`) | scripted |
| Mental health | CDC PLACES (`MHLTH`, census tract) | scripted |
| Transit | CTA 'L' station ridership + Divvy trips | scripted |

**Vancouver** (`notebooks/vancouver/`):

| Layer | Source | Access |
|-------|--------|--------|
| Crime | VPD GeoDASH open data (property crime; violent suppressed) | scripted |
| Boundaries | Statistics Canada 2021 Dissemination Areas | scripted |
| Deprivation | Canadian Index of Multiple Deprivation (CIMD) 2021 | scripted |
| Weather | Environment and Climate Change Canada (YVR) | scripted |

Two Chicago substitutions are documented in the code because the direct analogue is authentication-gated: CDC/ATSDR SVI is used in place of the login-gated Area Deprivation Index, and demographic counts come from the SVI release because the Census ACS API now requires a key.
